Ajouter du code JavaScript dans sa page HTML

La syntaxe de la balise script

Il y a deux façon pour ajouter un script sur le fichier HTML.

La première est d'ajouter directement le code entre la balise script.

                    <script>
                        // Ici on y mettra notre code JavaScript.
                    </script>

Et la deuxième est de passer par un fichier .js et de donner son chemin avec l'attribut src.

                    <script src="assets/js/nom-du-fichier.js"></script>

Le positionnement de la balise script sur le fichier HTML.

L'emplacement de la balise script est assez importante suivant les cas d'utilisation du script.

Si le script n'influe pas spécialement sur la page html. Alors elle pourra-être placé dans la balise head.

                    <!DOCTYPE html>

                    <html>

                      <head>

                        <title>Titre de la page</title>

                        <script src="assets/js/nom-du-fichier.js"></script>

                      </head>

                    </html>

Sinon et par principe elle sera plutôt placé en dernier dans la balise body.

                    <!DOCTYPE html>

                    <html>

                      <head></head>

                      <body>

                        <header></header>

                        <main></main>

                        <footer></footer>

                        <script src="assets/js/nom-du-fichier.js"></script>

                      </body>

                    </html>

Attention: dans cet exemple j'ai indiqué le src mais il n'est pas obligatoire suivant les cas de possitionnement. Il est bien sûr possible d'ajouter directement le code dans les balises script.
