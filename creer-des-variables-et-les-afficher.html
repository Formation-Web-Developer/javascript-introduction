La syntaxe pour déclarer une variable.

Pour bien commencé il est important de voir la bonne syntaxe pour déclarer sans problème une variable.

                    type name = value;

Il existe 3 différents types de variable

var, let et le const

Je ne rentrerais pas dans les détailles tout de suite mais sachez que chaque type correspond à une portabilité (Un mot compliqué pour l'instant). Contentons nous du var.

Ensuite le nom devra être écrit en camelCase et généralement en anglais sans accent, ni espace ou caractère spéciaux. Et c'est vous qui décidez de l'appeler comme vous le souhaitez. (Mieux vaut aussi qu'il soit en rapport avec la donnée qu'elle comportera.)

Pour les valeurs, nous les verrons un peu plus bas.

Voici un exemple de variable avec une bonne syntaxe

                    var name = 'John Doe';

PS: Comme vous le remarquez, il y a un point virgule à la fin. Sachez qu'il n'est pas obligatoire en JavaScript. Cependant nous prendrons l'habitude de le mettre. Cela indique une fin d'instruction, de plus il est souvent obligatoire dans d'autre langage de programmation telque PHP que l'on verra plus tard dans la formation.

Les variables peuvent aussi être déclarés sur une seule et même ligne et seront séparés par des virgules.

                    var name = 'Doe', lastname = 'John', age = 37;

Les valeurs possibles d'une variable.

En programmation nous avons le choix entres différents types de variables qui sont plus ou moins similaire mais contentons nous de JavaScript.

1- Les chaines de caractère

String: Ce type permet de déclarer une chaine de caractère. Elle sera entouré soit de guillemet simple ou de guillemet double ou bien de guillemet simple inversé.

Attention: Pour les guillemets simples inversés je vous le mets en exemple mais c'est comme pour les let et les const. Nous ne les utiliserons pas dans l'immédiat.

                    var name = 'Doe';

                    var lastname = "John";

                    var country = `France`;

Conseil: Si vous avez besoin d'ajouter un caractère dans votre chaine de caractère que vous utilisez pour englober votre chaine de caractère, vous devrez alors l'échapper avec /

                    var message = 'C\'est chez John ou l\'on trouve les meilleurs tartes à la crême.';

2- Les nombres entiers

Integer ou int: Ce type permet de déclarer un nombre entier. Ce qui signifie qu'il ne peut y avoir de décimale.

                    var age = 37;

3- Les nombres décimaux

Float ou Double: Ce type permet de déclarer un nombre qui accepte les décimales.

                    var money = 135.37;

Attention: Comme vous pourrez le constater, nous n'utilisons pas la virgule pour nos nombres floatant. Les langages de programmation ont étés faits sur la base de l'anglais. Par conséquent nous utlisons les points à la place des virgules.

Si jamais nous devons utiliser des gros nombres, nous ne pourrons pas non plus utiliser la virgule mais nous pouvons utiliser les underscore pour séparer en paquet de millier. C'est toujours plus jolie et plus compréhensible.

                    var money = 1_000_125.59;

                    //Au lieu de:

                    var money = 1000125.59;

3- Les valeurs booléennes.

Booléan: Ce type permet de déclarer une valeur qui sera soit vrai soit faux.

                    var value = true;

                    var value = false;

4- Les tableaux.

Array ou Collection: Ce type permet de déclarer une liste d'élément. Elle sera entouré de crochet et contiendra n'importe qu'elle type de valeur.

                    var items = ['Chien', 'Chat', 25, 4.369, true, false];

Pour récupérer un éléments d'un tableau nous utiliserons son index. C'est à dire le numéro de la case ou ce trouve l'objet en question entouré de crochet après le nom de la variable qui contient le tableau.

Attention: Les tableaux dans la plupart des langage de programmation ne commence pas par 1 mais par 0. Cela vaut aussi pour JavaScript. Par conséquent, si je souhaite récupérer la 3ème valeur de la liste, j'utiliserais l'index 2.

                    var items = ['Chien', 'Chat', 25, 4.369, true, false];

                    var item = items[2]; //item sera égale à 25.

5- Les tableaux à multi-dimension.

Alors cette partie est un bonus mais c'est quand même bon à savoir.

Il existe plusieurs dimension pour les tableaux. (De une à théoriquement une infinité.)

Le tableau que l'on a utilisé au dessus est un tableau par défaut. C'est à dire à une dimension. Si nous voulons plus de dimension alors il nous faudra ajouter des tableaux dans le tableaux initiale.

C'est confus ? Alors ne vous inquiétez pas c'est normal. Ce n'est pas évident à prendre en main mais une petite démonstration ne sera pas de trop:

                    var items = [

                      ['Chien', 'Chat']

                    ];

La je viens de créer un tableau à deux dimensions. C'est à dire, un tableau dans un tableau. Pour récupéré la valeur Chat, je devrais d'abord aller chercher mon tableau dans mon tableau initial. La ce n'est pas dur il est à la première position. Ensuite je pourrais récupérer ma valeur Chat qui se trouve à la seconde position.

En code ça donne ceci:

                    var items = [

                      ['Chien', 'Chat']

                    ];

                    var item = items[0]; // La je récupère mon tableau qui contient la valeur chien et chat.

                    var result = item[1]; // Et la je récupère ma valeur chat. 


                    //En plus court ça donnerai ça:

                    var result = items[0][1]; // Et la vous comprenez pourquoi on utilise le terme de dimension.

Comme vous le remarquez, nous avons utilisés deux fois les crochet pour récupéré notre valeur. C'est ainsi que l'on compte le nombre de dimension. En conclusion si nous avons un tableau à 3 dimensions alors nous aurions 3 fois des crochet pour récupérer notre valeur final.

Cela était surtout à but de démonstration. Ne vous prenez pas la tête sur ça ou bien demandez à vive voix pour plus d'explication.

6- Les objets

Object: Ce type permet de déclarer une collection de clé qui contiendront chacunes des valeurs différentes. L'object sera entouré d'accolade et chaques clés seront suivi du deux points et séparé par des virgules.

                    var user = {

                      name: 'Doe',

                      lastname: 'John',

                      age: 37,

                      country: 'France'

                    };

Pour appeler une variable dans un objet on utilisera le point comme séparateur.

                    var user = {

                      name: 'Doe',

                      lastname: 'John',

                      age: 37,

                      country: 'France'

                    };

                    var userName = user.name;

Afficher une variable dans la console.

Pour pouvoir afficher une variable dans la console il suffit d'utiliser l'objet console et d'appeler la fonction log puis de placer l'objet ciblé en paramètre de la fonction.

                    var name = 'John Doe';

                    console.log(name); // Afficher John Doe dans la console.

                    

                    var items = ['Chien', 'Chat'];

                    console.log(items); // Afficher le tableau dans la console.

                    console.log(items[1]); // Afficher Chat dans la console.
