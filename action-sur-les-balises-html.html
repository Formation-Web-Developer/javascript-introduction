Les actions sur les balises HTML

Ecrire du JavaScript dans les éléments.

Dans le chapitre précédent nous avons vu comment placer du code javascript dans une balise script. Mais il faut savoir qu'il est également possible d'écrire du JavaScript dans les éléments.

En utilisant un attribut dédié aux évenements de JavaScript.

                    <p onclick="prompt('Entrez une valeur');">Cliquez pour voir le prompt.</p>

Ou dans le href d'une balise avec en utilisant le préfix javascript: avant le code souhaitez.

                    <a href="javascript:prompt('Entrez une valeur')">Cliquez pour voir le prompt.</a>

La liste des événements possibles dans un attribut HTML

Il est possible d'utiliser plusieurs événements dans les attribut HTML.

                    onclick: Appelé lorsque l'on clique sur l'élément.

                    ondblclick: Appelé lorsque l'on double clique sur l'élément.

                    onmouseover: Appelé lorsque la souris survole l'élément.

                    onmouseout: Appelé lorsque la souris sort de l'élément.

                    onload: Appelé lorsque l'élément charge.

                    onunload: Appelé lorsque l'élément décharge.

Correction de l'exercice

Un exercice en deux partie nous avez était donné.

Le premier avait pour but de:

Mettre une balise img et lorsque l'on double clique sur celle-ci alors une alerte devait s'afficher avec le message Double Clique.

Pour cela il suffisait d'ajouter l'attribut ondblclick dans la balise img et utiliser la fonction alert() de JavaScript.

                    <img src="https://picsum.photos/300/300" ondblclick="alert('Double Clique');">

Rendu de l'exercice

Le second avait pour but de:

Ajouter un attribut pour détecter si la souris sortait de la balise img et y afficher une alerte avec le message La souris est sortie de l'image.

Pour cela il fallait ajouter l'attribut onmouseout à notre balise img et utiliser la fonction alert() de JavaScript.

                    <img src="https://picsum.photos/300/300" ondblclick="alert('Double Clique');" onmouseout="alert('La souris est sortie de l\'image');">
